# Beetle Website

這是一個基於 React 的網站專案。

## 資料夾結構

```
beetle_website/
├── public/                 # 靜態資源
├── src/
│   ├── components/         # 可重用的元件
│   ├── pages/              # 頁面
│   ├── context/            # 全域狀態管理
│   ├── utils/              # 工具函數 (API 請求等)
│   ├── App.jsx             # 應用主入口
│   ├── index.css           # 全域樣式
│   └── main.jsx            # React 渲染入口
├── .gitignore
├── package.json
└── README.md
```

## 啟動專案

1. 安裝依賴：
   ```
   npm install
   ```

2. 啟動開發伺服器：
   ```
   npm start
   ```
